Asintótica de retorno al origen

La difusión gaussiana 2D con simetría rotacional introduce el factor π: p₂ₙ(0,0) ∼ 1/(π·n)    (n→∞)

Estimador principal: log + 1/n sobre S̄ₙ

Sumando sobre n, la serie acumulada tiene estructura asintótica fina: S̄ₙ ≈ (1/π)·ln n + b + c/n Un OLS con tres regresores [ln n, 1, 1/n] estima directamente la pendiente â ≈ 1/π. Las constantes b y c absorben correcciones finitas. Este método es robusto porque estima una pendiente sobre una suma acumulada creciente, no invierte una media ruidosa. El IC se calcula desde los residuos del ajuste: δπ ≈ SE(â)/â².

Estimadores experimentales: OLS/WLS sobre n·p̂₂ₙ

Reescalar da una serie aproximadamente estacionaria: n · p̂₂ₙ(0,0) ≈ a + b/n + O(1/n²) Potencialmente más preciso asintóticamente, pero frágil: si por ruido â ≈ 0, la inversión 1/â explota. Se aplica un filtro de plausibilidad â ∈ [0.23, 0.44] antes de invertir. WLS usa pesos ∝ √n (suave).
La recurrencia en ℤ² garantiza retorno con probabilidad 1. Esta demo emplea la asintótica de Spitzer-Kac (1956) corregida por el kernel potencial de Fukai-Uchiyama (1996). R = 8 réplicas × N = 1000 caminantes. Los estimadores OLS/WLS son marcados como experimentales hasta que n sea suficientemente grande para estabilizar el intercepto.
