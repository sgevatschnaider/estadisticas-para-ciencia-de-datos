PROCESO GAUSSIANO + AUTÓMATA CELULAR

Idea central. El autómata celular genera la microdinámica local y discreta de la grilla. El proceso gaussiano no intenta modelar directamente el estado binario completo, sino reconstruir un campo macroscópico continuo derivado del autómata, en esta demo la densidad local de activación.

Micro vs. macro. Primero evoluciona el autómata con una regla local de propagación. Luego, para cada celda, se calcula una densidad local en una ventana vecina. Sobre un subconjunto observado de celdas se ajusta un GP espacial con kernel RBF o kernel basado en distancia de grafo.

Qué aporta el GP. Aporta suavizado espacial, interpolación en regiones no observadas, cuantificación explícita de incertidumbre y una geometría probabilística sobre la grilla. Por eso aparecen separados el campo real, la media posterior y el mapa de desviación estándar.

Aplicaciones. Esta metodología sirve para propagación de incendios, difusión epidémica, ocupación urbana, presión de uso del suelo, percolación, propagación de información, análisis de frentes de crecimiento y calibración de simuladores de dinámica discreta.

Lectura correcta de la demo. El mapa del autómata muestra el microestado binario. El mapa de densidad muestra el observable mesoscópico derivado del AC. La media GP reconstruye ese campo a partir de observaciones parciales. El mapa de incertidumbre marca dónde el modelo sabe menos.
