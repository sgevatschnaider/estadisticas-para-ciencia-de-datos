∂ Conceptos
Reverse diffusion: ruido ⟶ datos
Partimos de una nube gaussiana y, paso a paso, vamos empujando los puntos hacia regiones de alta densidad de una distribución objetivo.
No se entrena una red aquí: es una animación conceptual
En un modelo real, una red aprende la dirección de “desruido”. Aquí esa dirección se emula moviendo cada punto desde el ruido hacia la estructura final.
xt ≈ (1−p)·z + p·x0 + ruido residual
Cuando p aumenta, el peso del ruido cae y emerge la geometría del dato.
✦ Lectura didáctica
La idea central de los modelos de difusión es doble: primero se destruye la estructura de los datos mediante un proceso forward; luego se aprende a reconstruirla. Esta demo se concentra en la segunda parte. Empezamos con ruido gaussiano y lo vamos “ordenando” hacia una forma objetivo. No es un modelo entrenado: es una visualización conceptual del mapa ruido ⟶ datos.
